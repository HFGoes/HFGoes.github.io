    注意事项和细节

    1. 数组是多个相同类型的组合，实现对这些数据的统一管理
    2. 数组中的元素可以是任意数据类型，包括基本数据类型和引用的数据类型，但不能混用
    3. 数组创建后有默认值（String = null, boolean = false, char = \0000）
    4. 数组属于引用类型，数组行数据是对象(Object)

    值传递和引用传递的区别

    基本数据类型值传递时直接拷贝值；数组传递时是将地址拷贝，即arr1和arr2指向的地址相同，那么改变arr2的值的时候，arr1也会改变。

    排序简介

    排序时将多个数据，依指定的循序进行排列的过程
    排序的分类：

    1. 内部排序法：指将需要处理的所有数据都加载到内部存储器中进行排序。包括（交换式排序法、选择排序法和插入排序法）；
    2. 外部排序法：数据量过大，无法全部加载到内存中，需要借助外部存储进行排序。包括（合并排序法和直接合并排序法）。
