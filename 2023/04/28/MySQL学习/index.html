使用指令创建数据库

CREATE DATABASE gyf_db01 #创建一个名为gyf_01的数据库

如果没有指定字符集和校对规则，默认是utf8 和 utf8_general_ci(不区分大小写)

创建一个使用utf8字符集的gyf_db02

CREATE DATABASE gyf_db02 CHARACTER SET utf8

创建一个校对规则是utf8_bin的database

CREATE DATABASE gyf_db02 CHARACTER SET utf8 COLLATE utf8_bin

PS:在创建数据库，表的时候，为了规避关键字，可以用反引号解决(``)

DROP DATABASE gyf_db01
